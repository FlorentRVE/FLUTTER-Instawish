## Aperçu

### Le projet

Les objectifs du projet :

- Créer un profil
- Se connecter et gérer son profil
- Suivre d'autres profils
- Récupérer les posts des personnes que l'on suit
- Afficher des profils en mode "stories"
- Gérer ses posts
- Gestion des likes
